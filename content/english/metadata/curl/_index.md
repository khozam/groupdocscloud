          ```shell

            -H "Accept: application/json"

            curl -v "https://api.groupdocs.cloud/v1.0/metadata" \

            -H "Authorization: Bearer " \

                "FileInfo": {
                    "FilePath": "documents\\input.docx",
                    "StorageName": ""
                },
                "SearchCriteria": {
                    "ValueOptions": {

                }
          ```
